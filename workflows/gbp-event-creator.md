# GBP Event Creator Workflow

## Goal

Create approval-ready drafts for GBP Event, Offer, and Update posts.

## Inputs

- Google Sheets rows
- JSON payloads
- webhook payloads
- business config
- optional image assets or generated image prompts

## Outputs

- post type
- headline
- body copy
- CTA
- dates
- image recommendation
- duplicate check result
- approval record

## Safety

No post may be published until approved and live publishing is enabled.
