# Roadmap

## Phase 1: Framework

- repository structure
- business config templates
- reusable skills
- approval and logging design
- read-only workflow specifications

## Phase 2: Read Integrations

- authenticate connected Google account
- list GBP accounts and locations
- read listing details, reviews, posts, media, Q&A, and categories
- read Google Sheets post inputs

## Phase 3: Draft Generation

- optimizer reports
- event, offer, and update post drafts
- review reply drafts
- competitor comparison reports

## Phase 4: Approval System

- approval queue
- reviewer notes
- rejected draft handling
- audit trail

## Phase 5: Controlled Publishing

- publish approved posts
- reply to approved reviews
- update approved listing fields
- rollback and error handling

## Phase 6: Automation

- scheduled audits
- content calendars
- review monitoring
- competitor tracking
- business-specific extensions
