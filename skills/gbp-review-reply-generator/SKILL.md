# GBP Review Reply Generator

## Workflow

1. Load selected business config.
2. Pull recent reviews for the selected listing.
3. Identify sentiment, topics, praise, complaints, and recurring phrases.
4. Draft replies using the configured response style.
5. Avoid prohibited wording and unnatural keyword stuffing.
6. Save replies for approval.

## Output

- reply draft per review
- review theme summary
- SEO phrase suggestions
- operational improvement suggestions
