# GBP Optimizer

## Workflow

1. Load the selected business config.
2. Read the selected GBP listing data.
3. Analyze profile completeness, categories, services, description, hours, attributes, reviews, media, posts, and Q&A.
4. Compare findings against target keywords, prohibited wording, preferred CTAs, and posting frequency.
5. Produce a report with recommendations grouped by impact and effort.
6. Create approval-ready draft changes only where live updates would be useful.

## Output

- executive summary
- critical fixes
- quick wins
- content recommendations
- media recommendations
- suggested FAQs
- suggested posts
- approval items
